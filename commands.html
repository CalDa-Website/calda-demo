Moderation Commands

Commands for staff to moderate the server

Ban
Ban a member in your server
/ban @user
Kick
Kick a member in your server
/kick @user
Lockdown
Lockdown a channel in your server
/lockdown #channel
Clear
Deletes a specified number of messages for a channel or target.
/clear #channel @user
Nickname
Changes Nickname of a user.
/nickname @user
Slowmode
Slow downs the message rate in specified channel.
/slowdown #channel
Timeout
Timesout a user.
/timeout @user
/untimeout @user
Warnings
Add, Remove, Clear, Check Warnings of a user.
/warnings add @user
/warnings remove @user
/warnings check @user
/warnings clear @user

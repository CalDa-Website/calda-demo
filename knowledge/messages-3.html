Welcome message customization

Last updated on 2nd April 2021

Welcome messages are an essential part of server customization. Maki has a lot of customizable features you can use to make new members feel at home. You can edit all welcome messages on the dashboard under the welcome section or by using the welcome command.

Kinds of welcome messages
  • Regular message
  • Welcome image
  • Welcome embed
  • Direct message
